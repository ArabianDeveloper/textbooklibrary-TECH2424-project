## 📖 Overview
**Textbook Library** is a web-based application developed as part of the **TECH2424** course. It provides a platform to manage, organize, and browse a library of textbooks. The project is built using the robust Django framework (Python) for the backend, alongside HTML and CSS for the frontend interface.

## 🚀 Technologies Used
* **Backend:** Python, Django Web Framework
* **Frontend:** HTML5, CSS3
* **Database:** SQLite (Default Django database - `db.sqlite3`)
